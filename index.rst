    Si eres nuevo en `python`_, tal vez quieras empezar por darte  una idea de lo que
    es el lenguaje. Django es python  100%, así que si  te sientes un poco
    comodo con python probablemente sacaras mucho provecho de Django.

    Si eres completamente nuevo en  programación, puede que lo mejor seria comenzar con esta
    `lista de recursos pitón para no-programadores`_.

    Si ya sabes algunos otros lenguajes y quieres ponerte al dia con  python
    rápidamente, se recomienda inmersion en python (también disponible en
    version `dead-tree`_ ).
    Si no es absolutamente tu estilo, hay bastantes libros sobre python.


Esta es una pequeña lista que tal vez te puedan interesar todos estos libros estan en Español:

* El tutorial de python
