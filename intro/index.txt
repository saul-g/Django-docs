    Si eres nuevo usando `python`_, tal vez lo primero que necesitas es empezar por conocer este lenguaje de programación. Django es python 100%, así que si te sientes un poco cómodo con python probablemente sacaras mucho provecho de Django.

    Si eres completamente nuevo en  programación, puede que lo mejor seria 
    comenzar con esta `lista de recursos python para no-programadores`_.

    Si ya sabes o conoces algunos  otros lenguajes y quieres ponerte rápidamente al día con python, se recomienda leer el libro  **Inmersión en python**,  (para `Python 2`_, o para `Python 3`_, (también disponible en  versión `impresa`_ ). Si estos libros no son tu estilo, hay bastantes `libros sobre python`_ que te pueden ayudar a empezar.
